Atur tombol dan fitur apa saja yang muncul di toolbar penampil PDF.

## Semua Opsi

| Opsi | Tipe | Default | Deskripsi |
|--------|------|---------|-------------|
| `sidebar` | `boolean` | `true` | Tombol toggle sidebar thumbnail |
| `pageNavigation` | `boolean` | `true` | Kontrol navigasi halaman |
| `zoom` | `boolean` | `true` | Kontrol zoom |
| `search` | `boolean` | `true` | Alat pencarian |
| `rotate` | `boolean` | `true` | Tombol putar |
| `openFile` | `boolean` | `false` | Tombol buka file |
| `print` | `boolean` | `true` | Tombol cetak |
| `download` | `boolean` | `true` | Tombol unduh |
| `fullscreen` | `boolean` | `true` | Tombol layar penuh |
| `themeToggle` | `boolean` | `true` | Tombol ganti tema |
| `moreOptions` | `boolean` | `true` | Menu opsi lainnya |

## sidebar

Tampilkan/sembunyikan tombol toggle sidebar thumbnail.

```ts
toolbar: {
  sidebar: true
}
```

## pageNavigation

Tampilkan/sembunyikan kontrol navigasi halaman (tombol sebelumnya/berikutnya dan input halaman).

```ts
toolbar: {
  pageNavigation: true
}
```

## zoom

Tampilkan/sembunyikan kontrol zoom (perbesar, perkecil, pemilih level zoom).

```ts
toolbar: {
  zoom: true
}
```

## search

Tampilkan/sembunyikan fungsionalitas pencarian teks lengkap.

```ts
toolbar: {
  search: true
}
```

## rotate

Tampilkan/sembunyikan tombol putar.

```ts
toolbar: {
  rotate: true
}
```

## openFile

Tampilkan/sembunyikan tombol buka file. **Secara default dinonaktifkan** demi keamanan.

```ts
toolbar: {
  openFile: false
}
```

::warning
Mengaktifkan opsi ini memungkinkan pengguna membuka file PDF lokal apa pun. Aktifkan hanya jika memang diperlukan.
::

## print

Tampilkan/sembunyikan tombol cetak.

```ts
toolbar: {
  print: true
}
```

## download

Tampilkan/sembunyikan tombol unduh.

```ts
toolbar: {
  download: true
}
```

## fullscreen

Tampilkan/sembunyikan tombol toggle layar penuh.

```ts
toolbar: {
  fullscreen: true
}
```

## themeToggle

Tampilkan/sembunyikan tombol ganti tema terang/gelap.

```ts
toolbar: {
  themeToggle: true
}
```

## moreOptions

Tampilkan/sembunyikan menu dropdown "opsi lainnya".

```ts
toolbar: {
  moreOptions: true
}
```
