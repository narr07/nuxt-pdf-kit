Konfigurasi Nuxt PDF Kit secara global bisa kamu atur langsung di `nuxt.config.ts`.

## Konfigurasi Dasar

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-pdf-kit'],
  pdfKit: {
    toolbar: {
      // opsi toolbar
    },
  },
})
```

## Nilai Default

Semua opsi toolbar secara default sudah aktif, kecuali `openFile`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-pdf-kit'],
  pdfKit: {
    toolbar: {
      sidebar: true,
      pageNavigation: true,
      zoom: true,
      search: true,
      rotate: true,
      openFile: false,    // Nonaktif secara default
      print: true,
      download: true,
      fullscreen: true,
      themeToggle: true,
      moreOptions: true,
    },
  },
})
```
