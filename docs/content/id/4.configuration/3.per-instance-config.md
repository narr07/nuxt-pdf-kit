Override global configuration for specific `NuxtPdfKit` instances using the `toolbar` prop.

## Usage

```vue
<template>
  <!-- Uses global config -->
  <NuxtPdfKit src="/document1.pdf" />

  <!-- Overrides toolbar config -->
  <NuxtPdfKit
    src="/document2.pdf"
    :toolbar="{
      download: false,
      print: false,
    }"
  />
</template>
```

## Merging Behavior

Per-instance config merges with global config. Only specified options are overridden:

```ts [nuxt.config.ts]
// Global config
export default defineNuxtConfig({
  pdfKit: {
    toolbar: {
      search: true,
      download: true,
      print: true,
    },
  },
})
```

```vue
<!-- Per-instance override -->
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    download: false  // Only download is disabled
    // search and print remain true
  }"
/>
```

## Configuration Examples

### Minimal Viewer

```vue
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    sidebar: false,
    pageNavigation: true,
    zoom: true,
    search: false,
    rotate: false,
    print: false,
    download: false,
    fullscreen: false,
    themeToggle: false,
    moreOptions: false,
  }"
/>
```

### Read-Only Viewer

```vue
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    print: false,
    download: false,
    openFile: false,
  }"
/>
```

### Full-Featured Viewer

```vue
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    sidebar: true,
    pageNavigation: true,
    zoom: true,
    search: true,
    rotate: true,
    openFile: true,
    print: true,
    download: true,
    fullscreen: true,
    themeToggle: true,
    moreOptions: true,
  }"
/>
```
