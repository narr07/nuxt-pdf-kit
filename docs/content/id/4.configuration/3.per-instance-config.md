Kamu bisa menimpa (*override*) konfigurasi global untuk setiap instance `NuxtPdfKit` tertentu dengan menggunakan prop `toolbar`.

## Cara Penggunaan

```vue
<template>
  <!-- Menggunakan konfigurasi global -->
  <NuxtPdfKit src="/document1.pdf" />

  <!-- Menimpa konfigurasi toolbar -->
  <NuxtPdfKit
    src="/document2.pdf"
    :toolbar="{
      download: false,
      print: false,
    }"
  />
</template>
```

## Perilaku Penggabungan (*Merging*)

Konfigurasi per-instance akan digabungkan dengan konfigurasi global. Hanya opsi yang secara spesifik kamu tentukan yang akan ditimpa:

```ts [nuxt.config.ts]
// Konfigurasi global
export default defineNuxtConfig({
  pdfKit: {
    toolbar: {
      search: true,
      download: true,
      print: true,
    },
  },
})
```

```vue
<!-- Override per-instance -->
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    download: false  // Hanya download yang dinonaktifkan
    // search dan print tetap true (mengikuti global)
  }"
/>
```

## Contoh Konfigurasi

### Viewer Minimalis

```vue
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    sidebar: false,
    pageNavigation: true,
    zoom: true,
    search: false,
    rotate: false,
    print: false,
    download: false,
    fullscreen: false,
    themeToggle: false,
    moreOptions: false,
  }"
/>
```

### Viewer Hanya-Baca (*Read-Only*)

```vue
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    print: false,
    download: false,
    openFile: false,
  }"
/>
```

### Viewer Fitur Lengkap

```vue
<NuxtPdfKit
  src="/document.pdf"
  :toolbar="{
    sidebar: true,
    pageNavigation: true,
    zoom: true,
    search: true,
    rotate: true,
    openFile: true,
    print: true,
    download: true,
    fullscreen: true,
    themeToggle: true,
    moreOptions: true,
  }"
/>
```
