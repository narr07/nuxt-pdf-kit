Composable ini menyediakan fitur *virtual scrolling* untuk me-render dokumen berukuran besar dengan sangat efisien.

## Cara Penggunaan

```ts
const {
  visiblePages,
  isVirtualScrollEnabled,
  setContainerRef,
  scrollToPage,
} = usePdfKitVirtualScroll()
```

## Opsi

```ts
usePdfKitVirtualScroll({
  totalPages: 100,
  threshold: 10,
  overscan: 2,
})
```

| Opsi | Tipe | Default | Deskripsi |
|--------|------|---------|-------------|
| `totalPages` | `number` | `0` | Total jumlah halaman |
| `threshold` | `number` | `10` | Jumlah halaman minimum untuk mengaktifkan virtual scroll |
| `overscan` | `number` | `2` | Jumlah halaman tambahan yang di-render di luar viewport |

## Nilai Kembalian

| Properti | Tipe | Deskripsi |
|----------|------|-------------|
| `visiblePages` | `Ref<number[]>` | Daftar halaman yang akan di-render |
| `isVirtualScrollEnabled` | `ComputedRef<boolean>` | Status aktif fitur virtual scroll |
| `setContainerRef` | `(el: HTMLElement) => void` | Mengatur elemen kontainer scroll |
| `scrollToPage` | `(page: number) => void` | Gulir ke halaman tertentu |

## Contoh

```vue
<template>
  <div ref="containerRef" class="h-[600px] overflow-auto">
    <div v-for="page in visiblePages" :key="page">
      <!-- Render halaman {{ page }} -->
    </div>
  </div>
</template>

<script setup>
const containerRef = ref(null)

const {
  visiblePages,
  isVirtualScrollEnabled,
  setContainerRef,
} = usePdfKitVirtualScroll({
  totalPages: 100,
  threshold: 10,
})

onMounted(() => {
  if (containerRef.value) {
    setContainerRef(containerRef.value)
  }
})
</script>
```

## Cara Kerjanya

*Virtual scrolling* hanya akan me-render halaman-halaman yang terlihat di layar (*viewport*) plus beberapa halaman tambahan (*overscan*) agar pengalaman menggulir tetap mulus. Teknik ini secara signifikan meningkatkan performa saat membuka dokumen dengan jumlah halaman yang sangat banyak.

::tip
*Virtual scrolling* secara otomatis aktif di `NuxtPdfKit` ketika dokumen memiliki jumlah halaman melebihi batas *threshold* yang ditentukan.
::
