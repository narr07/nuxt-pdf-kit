Menyediakan berbagai utilitas yang kamu butuhkan untuk navigasi halaman.

## Cara Penggunaan

```ts
const {
  currentPage,
  totalPages,
  goToPage,
  nextPage,
  prevPage,
  firstPage,
  lastPage,
  canGoNext,
  canGoPrev,
} = usePdfKitPageNavigation()
```

## Opsi

```ts
usePdfKitPageNavigation({
  totalPages: 100,
  initialPage: 1,
})
```

| Opsi | Tipe | Default | Deskripsi |
|--------|------|---------|-------------|
| `totalPages` | `number` | `0` | Total jumlah halaman |
| `initialPage` | `number` | `1` | Nomor halaman awal |

## Nilai Kembalian

| Properti | Tipe | Deskripsi |
|----------|------|-------------|
| `currentPage` | `Ref<number>` | Nomor halaman saat ini |
| `totalPages` | `Ref<number>` | Total halaman |
| `goToPage` | `(page: number) => void` | Pindah ke halaman tertentu |
| `nextPage` | `() => void` | Lanjut ke halaman berikutnya |
| `prevPage` | `() => void` | Kembali ke halaman sebelumnya |
| `firstPage` | `() => void` | Ke halaman pertama |
| `lastPage` | `() => void` | Ke halaman terakhir |
| `canGoNext` | `ComputedRef<boolean>` | Cek apakah bisa ke halaman berikutnya |
| `canGoPrev` | `ComputedRef<boolean>` | Cek apakah bisa ke halaman sebelumnya |

## Contoh

```vue
<template>
  <div class="flex items-center gap-2">
    <UButton :disabled="!canGoPrev" @click="prevPage">
      Sebelumnya
    </UButton>
    <span>{{ currentPage }} / {{ totalPages }}</span>
    <UButton :disabled="!canGoNext" @click="nextPage">
      Berikutnya
    </UButton>
    <UInput
      type="number"
      :model-value="currentPage"
      @change="goToPage(Number($event.target.value))"
    />
  </div>
</template>

<script setup>
const {
  currentPage,
  totalPages,
  goToPage,
  nextPage,
  prevPage,
  canGoNext,
  canGoPrev,
} = usePdfKitPageNavigation({ totalPages: 50 })
</script>
```
