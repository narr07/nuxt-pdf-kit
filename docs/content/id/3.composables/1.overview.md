Nuxt PDF Kit menyediakan berbagai *composables* keren yang bisa kamu gunakan untuk mengontrol dokumen PDF secara programatik. Semua *composables* ini mengikuti standar terbaik modul Nuxt dan diawali dengan prefix `usePdfKit`.

## Daftar Composables

| Composable | Deskripsi |
|------------|-------------|
| `usePdfKitDocument` | Memuat dan mengelola dokumen PDF |
| `usePdfKitZoom` | Kontrol zoom dan pengaturan skala |
| `usePdfKitSearch` | Pencarian teks lengkap dengan fitur *highlighting* |
| `usePdfKitViewMode` | Mengatur mode tampilan dan mode gulir |
| `usePdfKitPageNavigation` | Utilitas navigasi halaman |
| `usePdfKitRotation` | Mengatur rotasi dokumen |
| `usePdfKitVirtualScroll` | *Virtual scrolling* untuk performa dokumen besar |

## Impor Otomatis

Semua *composables* sudah mendukung fitur *auto-import*. Jadi, kamu bisa langsung menggunakannya tanpa perlu repot-repot mengimpor manual:

```vue
<script setup>
const { scale, zoomIn, zoomOut } = usePdfKitZoom()
const { currentPage, nextPage, prevPage } = usePdfKitPageNavigation()
</script>
```

## Dukungan TypeScript

Dukungan TypeScript sudah tersedia secara penuh. Kamu bisa mengimpor tipe data jika diperlukan:

```ts
import type { ZoomLevel, ViewMode, ScrollMode, SearchMatch } from 'nuxt-pdf-kit'
```
