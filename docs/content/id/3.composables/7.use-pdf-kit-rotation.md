Manages document rotation.

## Usage

```ts
const {
  rotation,
  rotate,
  rotateClockwise,
  rotateCounterClockwise,
  resetRotation,
} = usePdfKitRotation()
```

## Return Values

| Property | Type | Description |
|----------|------|-------------|
| `rotation` | `Ref<number>` | Current rotation (0, 90, 180, 270) |
| `rotate` | `(degrees: number) => void` | Set rotation |
| `rotateClockwise` | `() => void` | Rotate +90째 |
| `rotateCounterClockwise` | `() => void` | Rotate -90째 |
| `resetRotation` | `() => void` | Reset to 0째 |

## Example

```vue
<template>
  <div class="flex items-center gap-2">
    <UButton @click="rotateCounterClockwise">
      <UIcon name="i-lucide-rotate-ccw" />
    </UButton>
    <span>{{ rotation }}째</span>
    <UButton @click="rotateClockwise">
      <UIcon name="i-lucide-rotate-cw" />
    </UButton>
    <UButton @click="resetRotation">Reset</UButton>
  </div>
</template>

<script setup>
const { 
  rotation, 
  rotateClockwise, 
  rotateCounterClockwise, 
  resetRotation 
} = usePdfKitRotation()
</script>
```
