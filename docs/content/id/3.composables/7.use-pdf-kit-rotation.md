Composable ini berfungsi untuk mengatur rotasi dokumen.

## Cara Penggunaan

```ts
const {
  rotation,
  rotate,
  rotateClockwise,
  rotateCounterClockwise,
  resetRotation,
} = usePdfKitRotation()
```

## Nilai Kembalian

| Properti | Tipe | Deskripsi |
|----------|------|-------------|
| `rotation` | `Ref<number>` | Rotasi saat ini (0, 90, 180, 270) |
| `rotate` | `(degrees: number) => void` | Mengatur rotasi |
| `rotateClockwise` | `() => void` | Putar +90째 (searah jarum jam) |
| `rotateCounterClockwise` | `() => void` | Putar -90째 (berlawanan jarum jam) |
| `resetRotation` | `() => void` | Reset ke 0째 |

## Contoh

```vue
<template>
  <div class="flex items-center gap-2">
    <UButton @click="rotateCounterClockwise">
      <UIcon name="i-lucide-rotate-ccw" />
    </UButton>
    <span>{{ rotation }}째</span>
    <UButton @click="rotateClockwise">
      <UIcon name="i-lucide-rotate-cw" />
    </UButton>
    <UButton @click="resetRotation">Reset</UButton>
  </div>
</template>

<script setup>
const { 
  rotation, 
  rotateClockwise, 
  rotateCounterClockwise, 
  resetRotation 
} = usePdfKitRotation()
</script>
```
