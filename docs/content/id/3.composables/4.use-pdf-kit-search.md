Menyediakan fungsionalitas pencarian teks lengkap dengan fitur penyorotan (*highlighting*) pada hasil yang ditemukan.

## Cara Penggunaan

```ts
const {
  searchQuery,
  searchMatches,
  currentMatchIndex,
  isSearching,
  caseSensitive,
  wholeWords,
  search,
  nextMatch,
  prevMatch,
  clearSearch,
  toggleCaseSensitive,
  toggleWholeWords,
} = usePdfKitSearch()
```

## Nilai Kembalian

| Properti | Tipe | Deskripsi |
|----------|------|-------------|
| `searchQuery` | `Ref<string>` | Query pencarian saat ini |
| `searchMatches` | `Ref<SearchMatch[]>` | Array hasil pencarian yang cocok |
| `currentMatchIndex` | `Ref<number>` | Indeks hasil pencarian saat ini |
| `isSearching` | `Ref<boolean>` | Status apakah sedang mencari |
| `caseSensitive` | `Ref<boolean>` | Mode pencarian sensitif huruf besar/kecil |
| `wholeWords` | `Ref<boolean>` | Mode pencarian kata utuh saja |
| `search` | `(query: string) => Promise<void>` | Melakukan pencarian |
| `nextMatch` | `() => void` | Lanjut ke hasil berikutnya |
| `prevMatch` | `() => void` | Kembali ke hasil sebelumnya |
| `clearSearch` | `() => void` | Membersihkan hasil pencarian |
| `toggleCaseSensitive` | `() => void` | Mengaktifkan/nonaktifkan sensitivitas huruf |
| `toggleWholeWords` | `() => void` | Mengaktifkan/nonaktifkan pencarian kata utuh |

## Interface SearchMatch

```ts
interface SearchMatch {
  pageIndex: number
  matchIndex: number
  text: string
}
```

## Contoh

```vue
<template>
  <div class="flex items-center gap-2">
    <UInput
      v-model="query"
      placeholder="Cari..."
      @keyup.enter="performSearch"
    />
    <span v-if="searchMatches.length">
      {{ currentMatchIndex + 1 }} / {{ searchMatches.length }}
    </span>
    <UButton @click="prevMatch">Sebelumnya</UButton>
    <UButton @click="nextMatch">Berikutnya</UButton>
    <UButton @click="clearSearch">Hapus</UButton>
  </div>
</template>

<script setup>
const query = ref('')
const {
  searchMatches,
  currentMatchIndex,
  search,
  nextMatch,
  prevMatch,
  clearSearch,
} = usePdfKitSearch()

async function performSearch() {
  await search(query.value)
}
</script>
```
