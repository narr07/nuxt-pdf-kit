## Instalasi Cepat

Cara termudah untuk menambahkan Nuxt PDF Kit ke aplikasi Nuxt Anda adalah menggunakan Nuxt CLI:

```bash [Terminal]
npx nuxt module add nuxt-pdf-kit

```

::tip
Perintah ini secara otomatis menambahkan modul ke `nuxt.config.ts` Anda dan menginstal paketnya.
::

## Instalasi Manual

Jika Anda lebih suka menginstal secara manual, ikuti langkah-langkah berikut:

::steps

### Instal paket

Menggunakan manajer paket pilihan Anda:

::code-group

```bash [npm]
npm i nuxt-pdf-kit

```

```bash [yarn]
yarn add nuxt-pdf-kit

```

```bash [pnpm]
pnpm add nuxt-pdf-kit

```

```bash [bun]
bun add nuxt-pdf-kit

```

::

### Tambahkan ke Konfigurasi Nuxt

Tambahkan `nuxt-pdf-kit` ke array `modules` di `nuxt.config.ts` Anda:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["nuxt-pdf-kit"],
});

```

### Mulai menggunakan penampil PDF

Anda siap menggunakan penampil PDF di komponen Anda:

```vue [app.vue]
<template>
  <NuxtPdfKit src="/sample.pdf" />
</template>

```

::

## Konfigurasi Modul

Anda dapat mengonfigurasi modul secara global di `nuxt.config.ts` Anda:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["nuxt-pdf-kit"],
  pdfKit: {
    // Konfigurasi Toolbar
    toolbar: {
      sidebar: true, // Toggle sidebar thumbnail
      pageNavigation: true, // Kontrol navigasi halaman
      zoom: true, // Kontrol zoom
      search: true, // Alat pencarian
      rotate: true, // Tombol putar
      openFile: false, // Tombol buka file (dinonaktifkan secara default)
      print: true, // Tombol cetak
      download: true, // Tombol unduh
      fullscreen: true, // Tombol layar penuh
      themeToggle: true, // Tombol toggle tema
      moreOptions: true, // Menu opsi lainnya
    },
  },
});

```

### Opsi Toolbar

::field-group
::field{name="sidebar" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan tombol toggle sidebar thumbnail.
::

::field{name="pageNavigation" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan kontrol navigasi halaman (tombol halaman sebelumnya/berikutnya dan input halaman).
::

::field{name="zoom" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan kontrol zoom (perbesar, perkecil, dan pemilih tingkat zoom).
::

::field{name="search" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan alat pencarian untuk fungsionalitas pencarian teks lengkap.
::

::field{name="rotate" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan tombol putar untuk rotasi dokumen.
::

::field{name="openFile" type="boolean" defaultValue="false"}
Tampilkan/sembunyikan tombol buka file (dinonaktifkan secara default untuk keamanan).
::

::field{name="print" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan tombol cetak untuk fungsionalitas cetak bawaan.
::

::field{name="download" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan tombol unduh untuk menyimpan PDF.
::

::field{name="fullscreen" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan tombol toggle layar penuh.
::

::field{name="themeToggle" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan tombol toggle tema (mode terang/gelap).
::

::field{name="moreOptions" type="boolean" defaultValue="true"}
Tampilkan/sembunyikan menu dropdown opsi lainnya.
::
::

## Dependensi yang Terinstal Otomatis

Nuxt PDF Kit secara otomatis mengelola dependensinya. Paket-paket berikut akan diinstal secara otomatis jika belum tersedia:

| Paket | Deskripsi |
| --- | --- |
| `@nuxt/ui` | Nuxt UI untuk komponen antarmuka premium |
| `@vueuse/nuxt` | Composables VueUse untuk fungsionalitas tambahan |
| `pdfjs-dist` | Pustaka PDF.js Mozilla untuk rendering PDF |

::note
Anda tidak perlu menginstal paket-paket ini secara manual. Nuxt PDF Kit menangani instalasi secara otomatis saat Anda menambahkan modul.
::

## Penggunaan Dasar

Setelah terinstal, Anda dapat menggunakan komponen `NuxtPdfKit` di mana saja dalam aplikasi Anda:

```vue
<template>
  <div class="h-screen">
    <NuxtPdfKit src="/document.pdf" />
  </div>
</template>

```

### Memuat dari URL

```vue
<template>
  <NuxtPdfKit src="[https://example.com/document.pdf](https://example.com/document.pdf)" />
</template>

```

### Memuat dari folder publik

Letakkan PDF Anda di folder `public/` dan referensikan secara langsung:

```vue
<template>
  <NuxtPdfKit src="/my-document.pdf" />
</template>

```
