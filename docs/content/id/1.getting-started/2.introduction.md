Selamat datang di **Nuxt PDF Kit**, modul penampil PDF kaya fitur untuk Nuxt dengan [Nuxt UI v3](https://ui.nuxt.com) dan [Tailwind CSS v4](https://tailwindcss.com).

## Apa itu Nuxt PDF Kit?

Nuxt PDF Kit adalah solusi penampil PDF komprehensif yang menyediakan semua yang Anda butuhkan untuk menampilkan, menavigasi, dan berinteraksi dengan dokumen PDF di aplikasi Nuxt Anda. Dibangun di atas [PDF.js](https://mozilla.github.io/pdf.js/), modul ini menawarkan antarmuka yang indah dan responsif siap pakai.

::card-group
:::card{icon="i-lucide-file-text" title="Rendering PDF"}
Rendering PDF berkualitas tinggi menggunakan pustaka PDF.js dari Mozilla.
:::

:::card{icon="i-lucide-search" title="Pencarian Teks"}
Cari teks di dokumen PDF dengan penyorotan dan navigasi.
:::

:::card{icon="i-lucide-palette" title="Nuxt UI"}
Antarmuka indah berbasis Nuxt UI v3 dan Tailwind CSS v4.
:::

:::card{icon="i-lucide-moon" title="Mode Gelap"}
Toggle tema terang/gelap bawaan dengan penyimpanan preferensi.
:::
::

## Mengapa Nuxt PDF Kit?

Membangun penampil PDF di Nuxt biasanya melibatkan:

- **Kompleksitas PDF.js** – Memahami dan mengonfigurasi pustaka PDF.js
- **Masalah SSR** – Menangani komponen khusus klien dan hidrasi
- **Pengembangan UI** – Menciptakan antarmuka responsif dan aksesibel dari nol
- **Implementasi fitur** – Menambahkan pencarian, zoom, cetak, unduh, dan lainnya

**Nuxt PDF Kit** menyelesaikan semua masalah ini dengan menyediakan:

::field-group
:::field{name="Nuxt UI" type="feature"}
Antarmuka indah dan responsif dengan dukungan mode terang/gelap siap pakai.
:::

:::field{name="Kompatibel SSR" type="feature"}
Bekerja mulus dengan server-side rendering Nuxt tanpa konfigurasi tambahan.
:::

:::field{name="Zero Config" type="feature"}
Otomatis memasang dependensi (@nuxt/ui, @vueuse/nuxt) – tinggal tambah dan pakai.
:::

:::field{name="Responsif" type="feature"}
Ramah seluler dengan toolbar adaptif yang bekerja di semua perangkat.
:::

:::field{name="Fitur Lengkap" type="feature"}
Pencarian, zoom, rotasi, cetak, unduh, thumbnail, pintasan keyboard, dan banyak lagi.
:::
::

## Fitur Utama

Nuxt PDF Kit menyertakan semua yang Anda butuhkan untuk pengalaman melihat PDF yang lengkap:

| Fitur                 | Deskripsi                                                    |
| --------------------- | ------------------------------------------------------------ |
| **Rendering PDF**     | Rendering berkualitas tinggi dengan PDF.js                   |
| **Pencarian Teks**    | Pencarian teks lengkap dengan penyorotan dan navigasi        |
| **Kontrol Zoom**      | Zoom masuk/keluar, pas halaman, pas lebar, persentase custom |
| **Rotasi**            | Rotasi dokumen searah jarum jam                              |
| **Mode Tampilan**     | Halaman tunggal, dua halaman, dua halaman dengan cover       |
| **Mode Scroll**       | Vertikal, horizontal, terbungkus, per halaman                |
| **Thumbnail**         | Sidebar dengan thumbnail halaman                             |
| **Pintasan Keyboard** | Navigasi dengan panah, zoom dengan +/-                       |
| **Mode Gelap**        | Toggle tema bawaan dengan penyimpanan preferensi             |
| **Cetak & Unduh**     | Dukungan cetak dan unduh native                              |
| **Virtual Scrolling** | Rendering efisien untuk dokumen besar                        |
| **Mode Responsif**    | Otomatis menyesuaikan halaman dengan lebar kontainer         |

## Persyaratan

Sebelum memulai, pastikan proyek Anda memenuhi persyaratan berikut:

- **Nuxt** >= 3.0.0
- **Node.js** >= 18

::tip
Nuxt PDF Kit secara otomatis memasang peer dependency yang dibutuhkan seperti `@nuxt/ui` dan `@vueuse/nuxt` jika belum ada di proyek Anda.
::

## Contoh Cepat

Berikut contoh sederhana untuk memulai:

```vue
<template>
  <NuxtPdfKit src="/sample.pdf" />
</template>
```

Itu saja! Anda sudah memiliki penampil PDF lengkap di aplikasi Nuxt.

Siap memulai? Lanjut ke panduan [Instalasi](/id/getting-started/installation).
