## Instalasi Cepat

Cara termudah menambahkan Nuxt PDF Kit ke aplikasi Nuxt Anda adalah menggunakan Nuxt CLI:

```bash [Terminal]
npx nuxi module add nuxt-pdf-kit
```

::tip
Perintah ini otomatis menambahkan modul ke `nuxt.config.ts` dan memasang paketnya.
::

## Instalasi Manual

Jika Anda lebih suka memasang secara manual, ikuti langkah berikut:

::steps
### Pasang paket

Gunakan manajer paket favorit Anda:

::tabs
  :::tabs-item{label="npm"}
  ```bash [Terminal]
  npm install nuxt-pdf-kit
  ```
  :::

  :::tabs-item{label="pnpm"}
  ```bash [Terminal]
  pnpm add nuxt-pdf-kit
  ```
  :::

  :::tabs-item{label="yarn"}
  ```bash [Terminal]
  yarn add nuxt-pdf-kit
  ```
  :::

  :::tabs-item{label="bun"}
  ```bash [Terminal]
  bun add nuxt-pdf-kit
  ```
  :::
::

### Tambahkan ke Nuxt Config

Tambahkan `nuxt-pdf-kit` ke array `modules` di `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-pdf-kit']
})
```

### Mulai gunakan penampil PDF

Sekarang Anda siap menggunakan penampil PDF di komponen:

```vue [app.vue]
<template>
  <NuxtPdfKit src="/sample.pdf" />
</template>
```
::

## Konfigurasi Modul

Anda dapat mengonfigurasi modul secara global di `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-pdf-kit'],
  pdfKit: {
    // Konfigurasi toolbar
    toolbar: {
      sidebar: true,        // Tombol toggle sidebar thumbnail
      pageNavigation: true, // Kontrol navigasi halaman
      zoom: true,           // Kontrol zoom
      search: true,         // Alat pencarian
      rotate: true,         // Tombol rotasi
      openFile: false,      // Tombol buka file (nonaktif secara default)
      print: true,          // Tombol cetak
      download: true,       // Tombol unduh
      fullscreen: true,     // Tombol layar penuh
      themeToggle: true,    // Tombol toggle tema
      moreOptions: true,    // Menu opsi lainnya
    },
  },
})
```

### Opsi Toolbar

::field-group
  ::field{name="sidebar" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan tombol toggle sidebar thumbnail.
  ::

  ::field{name="pageNavigation" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan kontrol navigasi halaman (tombol halaman sebelumnya/berikutnya dan input halaman).
  ::

  ::field{name="zoom" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan kontrol zoom (zoom in, zoom out, dan pemilih level zoom).
  ::

  ::field{name="search" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan alat pencarian untuk fungsionalitas pencarian teks.
  ::

  ::field{name="rotate" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan tombol rotasi dokumen.
  ::

  ::field{name="openFile" type="boolean" defaultValue="false"}
  Tampilkan/sembunyikan tombol buka file (dinonaktifkan secara default demi keamanan).
  ::

  ::field{name="print" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan tombol cetak untuk fungsionalitas cetak native.
  ::

  ::field{name="download" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan tombol unduh untuk menyimpan PDF.
  ::

  ::field{name="fullscreen" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan tombol layar penuh.
  ::

  ::field{name="themeToggle" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan tombol toggle tema (terang/gelap).
  ::

  ::field{name="moreOptions" type="boolean" defaultValue="true"}
  Tampilkan/sembunyikan menu dropdown opsi lainnya.
  ::
::

## Dependensi Otomatis

Nuxt PDF Kit secara otomatis mengelola dependensinya. Paket berikut akan otomatis dipasang jika belum ada:

| Paket | Deskripsi |
|---------|-------------|
| `@nuxt/ui` | Nuxt UI v3 untuk komponen antarmuka premium |
| `@vueuse/nuxt` | Komposables VueUse untuk fungsionalitas tambahan |
| `pdfjs-dist` | PDF.js dari Mozilla untuk rendering PDF |

::note
Anda tidak perlu memasang paket-paket ini secara manual. Nuxt PDF Kit menangani pemasangan secara otomatis ketika Anda menambahkan modul.
::

## Penggunaan Dasar

Setelah terpasang, Anda dapat menggunakan komponen `NuxtPdfKit` di mana saja dalam aplikasi Anda:

```vue
<template>
  <div class="h-screen">
    <NuxtPdfKit src="/document.pdf" />
  </div>
</template>
```

### Memuat dari URL

```vue
<template>
  <NuxtPdfKit src="https://example.com/document.pdf" />
</template>
```

### Memuat dari folder public

Tempatkan PDF Anda di folder `public/` dan referensikan langsung:

```vue
<template>
  <NuxtPdfKit src="/my-document.pdf" />
</template>
```
