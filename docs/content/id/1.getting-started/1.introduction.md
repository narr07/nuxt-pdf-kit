Selamat datang di **Nuxt PDF Kit**, modul penampil PDF lengkap untuk Nuxt yang dibangun dengan [Nuxt UI](https://ui.nuxt.com) dan [Tailwind CSS v4](https://tailwindcss.com).

## Apa itu Nuxt PDF Kit?

Nuxt PDF Kit adalah solusi komprehensif untuk menampilkan, menavigasi, dan berinteraksi dengan dokumen PDF dalam aplikasi Nuxt Anda. Ditenagai oleh [PDF.js](https://mozilla.github.io/pdf.js/), modul ini menawarkan antarmuka yang indah dan responsif yang siap pakai (*right out of the box*).

## Mengapa Memilih Nuxt PDF Kit?

Membangun penampil PDF di Nuxt mengharuskan Anda mengatasi tantangan berikut:

- **Kompleksitas PDF.js** - Memahami dan mengonfigurasi pustaka PDF.js
- **Masalah SSR** - Menangani komponen *client-only* dan proses *hydration*
- **Pengembangan UI** - Membuat antarmuka yang responsif dan aksesibel dari nol
- **Implementasi Fitur** - Menambahkan fitur seperti pencarian, *zoom*, cetak, unduh, dan lainnya

**Nuxt PDF Kit** mengatasi semua tantangan ini dengan menyediakan:

::field-group
  ::field{name="Nuxt UI" type="feature"}
  Antarmuka yang indah dan responsif dengan dukungan mode gelap/terang yang siap pakai.
  ::

  ::field{name="Kompatibel dengan SSR" type="feature"}
  Bekerja mulus dengan *server-side rendering* Nuxt, tanpa perlu konfigurasi apa pun.
  ::

  ::field{name="Tanpa Konfigurasi" type="feature"}
  Secara otomatis menginstal dependensi (@nuxt/ui, @vueuse/nuxt) - cukup tambahkan dan gunakan.
  ::

  ::field{name="Responsif" type="feature"}
  Ramah seluler (*mobile-friendly*), dengan *toolbar* adaptif yang berfungsi di semua perangkat.
  ::

 ::field{name="Fitur Lengkap" type="feature"}
 Fitur lengkap, termasuk pencarian, *zoom*, rotasi, cetak, unduh, *thumbnails*, pintasan *keyboard*, dan banyak lagi.
 ::
::

## Fitur Utama

Nuxt PDF Kit mencakup semua yang Anda butuhkan untuk pengalaman melihat PDF yang lengkap:

| Fitur | Deskripsi |
| :--- | :--- |
| **Rendering PDF** | Rendering berkualitas tinggi dengan PDF.js |
| **Pencarian Teks** | Pencarian teks lengkap dengan penyorotan dan navigasi |
| **Kontrol Zoom** | *Zoom in/out*, sesuaikan halaman, sesuaikan lebar, persentase kustom |
| **Rotasi** | Putar dokumen searah jarum jam |
| **Mode Tampilan** | Satu halaman, dua halaman, dua halaman dengan sampul |
| **Mode Gulir** | Vertikal, horizontal, *wrapped*, halaman-per-halaman |
| **Thumbnail** | *Sidebar* dengan *thumbnail* halaman |
| **Pintasan Keyboard** | Navigasi dengan tombol panah, *zoom* dengan +/- |
| **Mode Gelap** | Tombol tema bawaan dengan persistensi sesi |
| **Cetak & Unduh** | Dukungan cetak dan unduh bawaan (*native*) |
| **Virtual Scrolling** | Rendering efisien untuk dokumen besar |
| **Mode Responsif** | Penyesuaian halaman otomatis mengikuti lebar kontainer |

## Persyaratan

Sebelum memulai, pastikan proyek Anda memenuhi persyaratan berikut:

- **Nuxt** >= 3.0.0
- **Node.js** >= 18

::tip
Nuxt PDF Kit secara otomatis menginstal dependensi yang diperlukan, seperti [`@nuxt/ui`](https://ui.nuxt.com) dan [`@vueuse/nuxt`](https://vueuse.org), jika belum tersedia di proyek Anda.
::
