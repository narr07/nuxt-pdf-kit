## Referensi Props

| Prop | Tipe | Default | Deskripsi |
|------|------|---------|-------------|
| `src` | `string` | **wajib** | URL atau path ke file PDF |
| `theme` | `'light' \| 'dark'` | `'light'` | Tema awal |
| `responsive` | `boolean` | `true` | Sesuaikan halaman otomatis ke lebar kontainer |
| `initialViewMode` | `'single' \| 'dual' \| 'dual-cover'` | `'single'` | Mode tampilan awal |
| `initialScrollMode` | `'vertical' \| 'horizontal' \| 'wrapped' \| 'page'` | `'vertical'` | Mode scroll awal |
| `virtualScroll` | `boolean` | `true` | Aktifkan virtual scrolling |
| `virtualScrollThreshold` | `number` | `10` | Ambang jumlah halaman untuk virtual scroll |
| `toolbar` | `ToolbarOptions` | - | Timpa konfigurasi toolbar |

## src

**Wajib.** URL atau path ke file PDF.

```vue
<!-- Dari folder public -->
<NuxtPdfKit src="/sample.pdf" />

<!-- Dari URL eksternal -->
<NuxtPdfKit src="https://example.com/document.pdf" />
```

## theme

Tema awal untuk penampil PDF. Opsi: `'light'` atau `'dark'`.

```vue
<NuxtPdfKit src="/document.pdf" theme="dark" />
```

## responsive

Jika aktif, halaman otomatis menyesuaikan dengan lebar kontainer.

```vue
<NuxtPdfKit src="/document.pdf" :responsive="true" />
```

## initialViewMode

Mode tampilan awal untuk menampilkan halaman.

- `single`: Tampilkan satu halaman sekaligus
- `dual`: Tampilkan dua halaman berdampingan
- `dual-cover`: Dua halaman dengan halaman pertama sebagai cover

```vue
<NuxtPdfKit src="/document.pdf" initial-view-mode="dual" />
```

## initialScrollMode

Mode scroll awal untuk menavigasi dokumen.

- `vertical`: Scroll vertikal melalui halaman
- `horizontal`: Scroll horizontal melalui halaman
- `wrapped`: Tata halaman dalam grid
- `page`: Navigasi per halaman

```vue
<NuxtPdfKit src="/document.pdf" initial-scroll-mode="horizontal" />
```

## virtualScroll

Aktifkan virtual scrolling untuk performa lebih baik pada dokumen besar.

```vue
<NuxtPdfKit src="/document.pdf" :virtual-scroll="true" />
```

## virtualScrollThreshold

Ambang jumlah halaman untuk mengaktifkan virtual scrolling.

```vue
<NuxtPdfKit src="/document.pdf" :virtual-scroll-threshold="20" />
```

## toolbar

Timpa konfigurasi toolbar global untuk instance tertentu.

```vue
<NuxtPdfKit 
  src="/document.pdf" 
  :toolbar="{
    search: true,
    print: false,
    download: true
  }"
/>
```
