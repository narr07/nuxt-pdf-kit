Komponen `NuxtPdfKit` menyediakan metode melalui template ref untuk kontrol terprogram.

## Penggunaan

```vue
<template>
  <NuxtPdfKit ref="pdfViewer" src="/document.pdf" />
  <button @click="goToPage5">Go to Page 5</button>
</template>

<script setup>
const pdfViewer = ref(null)

function goToPage5() {
  pdfViewer.value?.goToPage(5)
}
</script>
```

## Metode Tersedia

| Metode | Parameter | Deskripsi |
|--------|------------|-------------|
| `goToPage` | `(page: number)` | Navigasi ke halaman tertentu |
| `nextPage` | - | Ke halaman berikutnya |
| `prevPage` | - | Ke halaman sebelumnya |
| `zoomIn` | - | Perbesar tingkat zoom |
| `zoomOut` | - | Perkecil tingkat zoom |
| `setZoom` | `(level: number \| ZoomLevel)` | Atur tingkat zoom tertentu |
| `rotate` | - | Rotasi dokumen searah jarum jam |
| `print` | - | Buka dialog cetak |
| `download` | - | Unduh PDF |

## goToPage

Navigasi ke nomor halaman tertentu.

```vue
<script setup>
const pdfViewer = ref(null)

// Go to page 10
pdfViewer.value?.goToPage(10)
</script>
```

## nextPage / prevPage

Navigasi ke halaman berikutnya atau sebelumnya.

```vue
<script setup>
const pdfViewer = ref(null)

// Navigation
pdfViewer.value?.nextPage()
pdfViewer.value?.prevPage()
</script>
```

## zoomIn / zoomOut

Perbesar atau perkecil tingkat zoom.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.zoomIn()
pdfViewer.value?.zoomOut()
</script>
```

## setZoom

Atur tingkat atau mode zoom tertentu.

```vue
<script setup>
const pdfViewer = ref(null)

// Set to 150%
pdfViewer.value?.setZoom(1.5)

// Fit to page
pdfViewer.value?.setZoom('page-fit')

// Fit to width
pdfViewer.value?.setZoom('page-width')
</script>
```

## rotate

Rotasi dokumen searah jarum jam sebesar 90 derajat.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.rotate()
</script>
```

## print

Buka dialog cetak bawaan.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.print()
</script>
```

## download

Unduh file PDF.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.download()
</script>
```

```vue
      <UButton @click="pdfViewer?.zoomOut()">Zoom Out</UButton>

      <NuxtPdfKit ref="pdfViewer" src="/document.pdf" />
    </div>
  </div>

<script setup>

</script>
```
