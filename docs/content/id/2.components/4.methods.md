Komponen `NuxtPdfKit` menyediakan berbagai metode melalui *template ref* yang bisa kamu gunakan untuk kontrol secara programatik.

## Cara Menggunakan

```vue
<template>
  <NuxtPdfKit ref="pdfViewer" src="/document.pdf" />
  <button @click="goToPage5">Ke Halaman 5</button>
</template>

<script setup>
const pdfViewer = ref(null)

function goToPage5() {
  pdfViewer.value?.goToPage(5)
}
</script>
```

## Metode yang Tersedia

| Metode | Parameter | Deskripsi |
|--------|------------|-------------|
| `goToPage` | `(page: number)` | Pindah ke halaman tertentu |
| `nextPage` | - | Lanjut ke halaman berikutnya |
| `prevPage` | - | Kembali ke halaman sebelumnya |
| `zoomIn` | - | Memperbesar tampilan (zoom in) |
| `zoomOut` | - | Memperkecil tampilan (zoom out) |
| `setZoom` | `(level: number \| ZoomLevel)` | Mengatur level zoom secara spesifik |
| `rotate` | - | Memutar dokumen searah jarum jam |
| `print` | - | Membuka dialog pencetakan |
| `download` | - | Mengunduh file PDF |

## goToPage

Pindah langsung ke nomor halaman yang kamu inginkan.

```vue
<script setup>
const pdfViewer = ref(null)

// Pergi ke halaman 10
pdfViewer.value?.goToPage(10)
</script>
```

## nextPage / prevPage

Navigasi ke halaman selanjutnya atau sebelumnya dengan mudah.

```vue
<script setup>
const pdfViewer = ref(null)

// Navigasi
pdfViewer.value?.nextPage()
pdfViewer.value?.prevPage()
</script>
```

## zoomIn / zoomOut

Menambah atau mengurangi level zoom dokumen.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.zoomIn()
pdfViewer.value?.zoomOut()
</script>
```

## setZoom

Mengatur level zoom atau mode tampilan tertentu sesuai kebutuhanmu.

```vue
<script setup>
const pdfViewer = ref(null)

// Atur zoom ke 150%
pdfViewer.value?.setZoom(1.5)

// Sesuaikan dengan halaman (Page Fit)
pdfViewer.value?.setZoom('page-fit')

// Sesuaikan dengan lebar (Page Width)
pdfViewer.value?.setZoom('page-width')
</script>
```

## rotate

Memutar dokumen 90 derajat searah jarum jam.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.rotate()
</script>
```

## print

Membuka dialog pencetakan bawaan browser.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.print()
</script>
```

## download

Mengunduh file PDF yang sedang dibuka.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.download()
</script>
```

## Contoh Lengkap

```vue
<template>
  <div class="flex flex-col gap-4">
    <!-- Kontrol Kustom -->
    <div class="flex gap-2">
      <UButton @click="pdfViewer?.prevPage()">Sebelumnya</UButton>
      <UButton @click="pdfViewer?.nextPage()">Berikutnya</UButton>
      <UButton @click="pdfViewer?.zoomIn()">Perbesar</UButton>
      <UButton @click="pdfViewer?.zoomOut()">Perkecil</UButton>
      <UButton @click="pdfViewer?.rotate()">Putar</UButton>
      <UButton @click="pdfViewer?.print()">Cetak</UButton>
      <UButton @click="pdfViewer?.download()">Unduh</UButton>
    </div>

    <!-- PDF Viewer -->
    <div class="h-[600px]">
      <NuxtPdfKit ref="pdfViewer" src="/document.pdf" />
    </div>
  </div>
</template>

<script setup>
const pdfViewer = ref(null)
</script>
```
