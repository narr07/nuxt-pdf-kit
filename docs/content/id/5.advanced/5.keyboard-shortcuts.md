Nuxt PDF Kit menyertakan pintasan keyboard bawaan.

## Navigasi

| Tombol | Aksi |
|-----|--------|
| `←` atau `↑` | Halaman sebelumnya |
| `→` atau `↓` | Halaman berikutnya |
| `Home` | Halaman pertama |
| `End` | Halaman terakhir |
| `Page Up` | Halaman sebelumnya |
| `Page Down` | Halaman berikutnya |

## Zoom

| Tombol | Aksi |
|-----|--------|
| `+` atau `=` | Perbesar (Zoom in) |
| `-` | Perkecil (Zoom out) |
| `0` | Reset zoom |

## Pencarian

| Tombol | Aksi |
|-----|--------|
| `Ctrl/Cmd + F` | Buka pencarian |
| `Enter` | Hasil berikutnya |
| `Shift + Enter` | Hasil sebelumnya |
| `Escape` | Bersihkan pencarian |

## Lainnya

| Tombol | Aksi |
|-----|--------|
| `Escape` | Keluar mode layar penuh |
| `R` | Putar searah jarum jam |

## Tips Penggunaan

::tip
Klik pada viewer untuk memfokuskannya sebelum menggunakan pintasan keyboard.
::
