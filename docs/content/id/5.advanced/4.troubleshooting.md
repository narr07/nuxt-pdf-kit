Solusi untuk masalah umum saat menggunakan Nuxt PDF Kit.

## PDF Gagal Dimuat

**Periksa path:**
```vue
<!-- Correct: from public/ folder -->
<NuxtPdfKit src="/document.pdf" />

<!-- Wrong -->
<NuxtPdfKit src="public/document.pdf" />
```

**Masalah CORS:** Saat memuat dari URL eksternal, pastikan server mengizinkan permintaan lintas-origin.

## Teks Buram

Coba tingkatkan level zoom atau gunakan mode zoom `page-width`.

## Lambat Saat Memuat

Aktifkan virtual scrolling untuk dokumen besar:

```vue
<NuxtPdfKit 
  src="/large-document.pdf"
  :virtual-scroll="true"
  :virtual-scroll-threshold="10"
/>
```

## Error SSR

PDF viewer hanya berjalan di sisi klien. Jika menggunakan wrapper kustom, pastikan juga bersifat client-only:

```vue
<template>
  <ClientOnly>
    <YourWrapper>
      <NuxtPdfKit src="/document.pdf" />
    </YourWrapper>
  </ClientOnly>
</template>
```

## Pencarian Tidak Menemukan Teks

- **PDF hasil scan:** Jika PDF berupa gambar hasil scan, tidak ada teks yang bisa dicari.
- **Sensitivitas huruf besar/kecil:** Aktif/nonaktifkan opsi case sensitive di pengaturan pencarian.

## Cetak Menampilkan Halaman Kosong

Pratinjau cetak browser bergantung pada konten yang sudah dirender. Jika layout Anda menerapkan gaya yang membatasi, pratinjau bisa kosong atau terpotong.

- Pastikan semua halaman selesai dirender sebelum memanggil `window.print()`.
- Hindari overlay yang menutup konten (mis. modal fixed) saat mencetak.
- Pastikan `html, body` mengizinkan overflow saat mencetak.

```css [assets/css/main.css]
@media print {
  /* Allow full document flow */
  html, body {
    overflow: visible !important;
    height: auto !important;
  }

  /* Hide viewer UI chrome if needed */
  .npk-viewer .npk-toolbar,
  .npk-viewer .npk-main,
  .npk-viewer .npk-container {
    display: none !important;
  }

  /* Show your print container */
  .npk-print-container {
    display: block !important;
    position: absolute;
    left: 0;
    top: 0;
    width: 100%;
  }
}
```

Jika pratinjau masih bermasalah, sebagai solusi sementara Anda dapat membuka PDF di tab baru dan menggunakan fungsi cetak bawaan browser.
