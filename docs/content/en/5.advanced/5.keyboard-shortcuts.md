Nuxt PDF Kit includes built-in keyboard shortcuts.

## Navigation

| Key | Action |
|-----|--------|
| `←` or `↑` | Previous page |
| `→` or `↓` | Next page |
| `Home` | First page |
| `End` | Last page |
| `Page Up` | Previous page |
| `Page Down` | Next page |

## Zoom

| Key | Action |
|-----|--------|
| `+` or `=` | Zoom in |
| `-` | Zoom out |
| `0` | Reset zoom |

## Search

| Key | Action |
|-----|--------|
| `Ctrl/Cmd + F` | Open search |
| `Enter` | Next match |
| `Shift + Enter` | Previous match |
| `Escape` | Clear search |

## Other

| Key | Action |
|-----|--------|
| `Escape` | Exit fullscreen |
| `R` | Rotate clockwise |

## Usage Tips

::tip
Click on the viewer to focus it before using keyboard shortcuts.
::
