The `NuxtPdfKit` component exposes methods through a template ref for programmatic control.

## Usage

```vue
<template>
  <NuxtPdfKit ref="pdfViewer" src="/document.pdf" />
  <button @click="goToPage5">Go to Page 5</button>
</template>

<script setup>
const pdfViewer = ref(null)

function goToPage5() {
  pdfViewer.value?.goToPage(5)
}
</script>
```

## Available Methods

| Method | Parameters | Description |
|--------|------------|-------------|
| `goToPage` | `(page: number)` | Navigate to a specific page |
| `nextPage` | - | Go to the next page |
| `prevPage` | - | Go to the previous page |
| `zoomIn` | - | Increase zoom level |
| `zoomOut` | - | Decrease zoom level |
| `setZoom` | `(level: number \| ZoomLevel)` | Set specific zoom level |
| `rotate` | - | Rotate document clockwise |
| `print` | - | Open print dialog |
| `download` | - | Download the PDF |

## goToPage

Navigate to a specific page number.

```vue
<script setup>
const pdfViewer = ref(null)

// Go to page 10
pdfViewer.value?.goToPage(10)
</script>
```

## nextPage / prevPage

Navigate to the next or previous page.

```vue
<script setup>
const pdfViewer = ref(null)

// Navigation
pdfViewer.value?.nextPage()
pdfViewer.value?.prevPage()
</script>
```

## zoomIn / zoomOut

Increase or decrease the zoom level.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.zoomIn()
pdfViewer.value?.zoomOut()
</script>
```

## setZoom

Set a specific zoom level or mode.

```vue
<script setup>
const pdfViewer = ref(null)

// Set to 150%
pdfViewer.value?.setZoom(1.5)

// Fit to page
pdfViewer.value?.setZoom('page-fit')

// Fit to width
pdfViewer.value?.setZoom('page-width')
</script>
```

## rotate

Rotate the document clockwise by 90 degrees.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.rotate()
</script>
```

## print

Open the native print dialog.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.print()
</script>
```

## download

Download the PDF file.

```vue
<script setup>
const pdfViewer = ref(null)

pdfViewer.value?.download()
</script>
```
