## Props Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | **required** | URL or path to PDF file |
| `theme` | `'light' \| 'dark'` | `'light'` | Initial theme |
| `responsive` | `boolean` | `true` | Auto-fit pages to container |
| `initialViewMode` | `'single' \| 'dual' \| 'dual-cover'` | `'single'` | Initial view mode |
| `initialScrollMode` | `'vertical' \| 'horizontal' \| 'wrapped' \| 'page'` | `'vertical'` | Initial scroll mode |
| `virtualScroll` | `boolean` | `true` | Enable virtual scrolling |
| `virtualScrollThreshold` | `number` | `10` | Pages threshold for virtual scroll |
| `toolbar` | `ToolbarOptions` | - | Override toolbar config |

## src

**Required.** The URL or path to the PDF file.

```vue
<!-- From public folder -->
<NuxtPdfKit src="/sample.pdf" />

<!-- From external URL -->
<NuxtPdfKit src="https://example.com/document.pdf" />
```

## theme

Initial theme for the PDF viewer. Options: `'light'` or `'dark'`.

```vue
<NuxtPdfKit src="/document.pdf" theme="dark" />
```

## responsive

When enabled, automatically fits pages to the container width.

```vue
<NuxtPdfKit src="/document.pdf" :responsive="true" />
```

## initialViewMode

Initial view mode for displaying pages.

- `single`: Display one page at a time
- `dual`: Display two pages side by side
- `dual-cover`: Display two pages with first page as cover

```vue
<NuxtPdfKit src="/document.pdf" initial-view-mode="dual" />
```

## initialScrollMode

Initial scroll mode for navigating the document.

- `vertical`: Scroll vertically through pages
- `horizontal`: Scroll horizontally through pages
- `wrapped`: Wrap pages in a grid layout
- `page`: Navigate page by page

```vue
<NuxtPdfKit src="/document.pdf" initial-scroll-mode="horizontal" />
```

## virtualScroll

Enable virtual scrolling for better performance with large documents.

```vue
<NuxtPdfKit src="/document.pdf" :virtual-scroll="true" />
```

## virtualScrollThreshold

Number of pages threshold to enable virtual scrolling.

```vue
<NuxtPdfKit src="/document.pdf" :virtual-scroll-threshold="20" />
```

## toolbar

Override the global toolbar configuration for this specific instance.

```vue
<NuxtPdfKit 
  src="/document.pdf" 
  :toolbar="{
    search: true,
    print: false,
    download: true
  }"
/>
```
